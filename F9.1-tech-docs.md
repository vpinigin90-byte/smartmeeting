# Техническая документация: F9.1 - Step Navigation

**Версия:** 1.1  
**Компонент:** `BookingStepFlow`  
**Файл:** `booking-step-flow.jsx`  

## Описание

Многошаговая форма бронирования, объединяющая все компоненты в единый flow.

## Шаги

1. **Выбор длительности** (30/45/60 мин)
2. **Выбор даты и времени** (календарь)
3. **Заполнение контактов** (форма)
4. **Вопросы к встрече** (после контактов)

## State

```javascript
{
  currentStep: 1,           // Текущий шаг (1-4)
  bookingData: {
    duration: 60,           // Выбранная длительность
    date: null,             // Выбранная дата
    formData: null,         // Контактные данные
    questions: ''           // Вопросы после контактов
  }
}
```

## Функции

### handleDurationSelect(duration)
- Сохраняет длительность
- Переходит к шагу 2

### handleDateSelect(date)
- Сохраняет дату
- Переходит к шагу 3

### handleFormSubmit(formData)
- Сохраняет данные формы
- Переходит к шагу 4 (вопросы)

### handleQuestionsSubmit(questions)
- Сохраняет вопросы
- Завершает бронирование (отправка/подтверждение)

### goBack()
- Возврат на предыдущий шаг
- Сохраняет выбранные данные

## UI Элементы

### Progress Bar
- Показывает 4 шага
- Текущий шаг выделен синим
- Пройденные шаги с галочкой ✓
- Анимация прогресса

### Summary Panel
- Показывается на шагах 2, 3 и 4
- Отображает выбранные данные
- ⏱️ Длительность
- 📅 Дата (если выбрана)
- ❓ Индикатор добавленных вопросов

### Back Button
- Показывается на шагах 2, 3 и 4
- Возврат без потери данных

## Интеграция компонентов

```javascript
// Шаг 1
<DurationSelector onDurationChange={handleDurationSelect} />

// Шаг 2
<BookingCalendar 
  duration={bookingData.duration}
  onDateSelect={handleDateSelect}
/>

// Шаг 3
<BookingForm
  selectedDuration={bookingData.duration}
  selectedDate={bookingData.date}
  onSubmit={handleFormSubmit}
/>

// Шаг 4
<QuestionsStep
  value={bookingData.questions}
  onSubmit={handleQuestionsSubmit}
/>
```

## Анимации

- fadeIn - появление контента
- slideUp - появление summary панели
- Плавные переходы прогресс-бара
