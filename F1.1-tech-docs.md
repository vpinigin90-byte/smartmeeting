## 1. Описание функции

### 1.1 Назначение
Интерактивный календарь для выбора даты встречи с учетом:
- Производственного календаря РФ 2026
- Выходных дней (суббота, воскресенье)
- Прошедших дат
- Текущей даты (выделение)

### 1.2 Функциональные требования
- ✅ Отображение текущего месяца при загрузке
- ✅ Навигация между месяцами (← →)
- ✅ Выделение сегодняшней даты
- ✅ Блокировка прошедших дат
- ✅ Показ всех дней недели (Пн-Вс)
- ✅ Визуальное различие типов дней (рабочий/выходной/праздник)
- ✅ Выбор даты кликом
- ✅ Кнопка "Продолжить" после выбора

## 4. Основные функции

### 4.1 `getDaysInMonth(date)`

**Назначение:** Генерация массива дней для отображения в календаре

**Параметры:**
- `date` (Date) - дата месяца, который нужно отобразить

**Возвращает:** 
- `Array<Date | null>` - массив дат месяца с null для выравнивания

**Алгоритм:**
1. Определяет первый день месяца
2. Определяет последний день месяца
3. Вычисляет offset (сколько пустых ячеек нужно в начале)
4. Добавляет null для выравнивания
5. Добавляет все дни месяца (1-28/29/30/31)

**Пример:**
```javascript
// Для апреля 2026 (начинается со среды)
// Возвращает: [null, null, Date(1), Date(2), ..., Date(30)]
const days = getDaysInMonth(new Date(2026, 3, 1));
```

**Особенности:**
- Первый день недели = Понедельник (ISO 8601)
- Воскресенье (0) конвертируется в offset = 6
- Все 7 дней недели включены (включая выходные)

---

### 4.2 `isHoliday(date)`

**Назначение:** Проверка, является ли дата праздничным днем

**Параметры:**
- `date` (Date) - проверяемая дата

**Возвращает:** 
- `boolean` - true если праздник, false если нет

**Алгоритм:**
```javascript
const isHoliday = (date) => {
  if (!date) return false;
  const dateStr = date.toISOString().split('T')[0]; // 'YYYY-MM-DD'
  return holidays2026.includes(dateStr);
};
```

**Источник данных:** 
- Производственный календарь РФ 2026
- Хардкод массив `holidays2026`

---

### 4.3 `isWeekend(date)`

**Назначение:** Проверка, является ли дата выходным (Сб/Вс)

**Параметры:**
- `date` (Date) - проверяемая дата

**Возвращает:** 
- `boolean` - true если выходной, false если нет

**Алгоритм:**
```javascript
const isWeekend = (date) => {
  if (!date) return false;
  const dayOfWeek = date.getDay(); // 0 = Вс, 6 = Сб
  return dayOfWeek === 0 || dayOfWeek === 6;
};
```

---

### 4.4 `isPastDate(date)`

**Назначение:** Проверка, является ли дата прошедшей

**Параметры:**
- `date` (Date) - проверяемая дата

**Возвращает:** 
- `boolean` - true если прошедшая, false если нет

**Алгоритм:**
```javascript
const isPastDate = (date) => {
  if (!date) return false;
  const today = new Date();
  today.setHours(0, 0, 0, 0); // Сброс времени для сравнения только дат
  date.setHours(0, 0, 0, 0);
  return date < today;
};
```

**Важно:** Сравнение без учета времени (только даты)

---

### 4.5 `isToday(date)`

**Назначение:** Проверка, является ли дата сегодняшней

**Параметры:**
- `date` (Date) - проверяемая дата

**Возвращает:** 
- `boolean` - true если сегодня, false если нет

**Алгоритм:**
```javascript
const isToday = (date) => {
  if (!date) return false;
  const today = new Date();
  return (
    date.getDate() === today.getDate() &&
    date.getMonth() === today.getMonth() &&
    date.getFullYear() === today.getFullYear()
  );
};
```

---

### 4.6 `isSelected(date)`

**Назначение:** Проверка, является ли дата выбранной

**Параметры:**
- `date` (Date) - проверяемая дата

**Возвращает:** 
- `boolean` - true если выбрана, false если нет

**Алгоритм:**
```javascript
const isSelected = (date) => {
  if (!date || !selectedDate) return false;
  return (
    date.getDate() === selectedDate.getDate() &&
    date.getMonth() === selectedDate.getMonth() &&
    date.getFullYear() === selectedDate.getFullYear()
  );
};
```

---

### 4.7 `handleDateSelect(date)`

**Назначение:** Обработчик выбора даты пользователем

**Параметры:**
- `date` (Date) - выбранная дата

**Возвращает:** 
- `void`

**Алгоритм:**
```javascript
const handleDateSelect = (date) => {
  // Проверка доступности даты
  if (!date || isPastDate(date) || isToday(date) || isWeekend(date) || isHoliday(date)) {
    return; // Игнорируем клик
  }
  
  // Устанавливаем выбранную дату
  setSelectedDate(date);
};
```

**Условия блокировки:**
- Дата = null (пустая ячейка)
- Прошедшая дата
- **Сегодняшняя дата** (команде нужно время на подготовку к встрече)
- Выходной день
- Праздничный день

**Минимальная дата для записи:** Завтра (today + 1 день)

**Обоснование:** Команда Scrolltool (методисты и менеджер) не смогут быстро сориентироваться и подготовиться к встрече в день обращения. Требуется минимум 1 день на подготовку.

---

### 4.8 `goToPreviousMonth()`

**Назначение:** Переключение на предыдущий месяц

**Параметры:** нет

**Возвращает:** `void`

**Алгоритм:**
```javascript
const goToPreviousMonth = () => {
  setCurrentDate(
    new Date(currentDate.getFullYear(), currentDate.getMonth() - 1)
  );
};
```

**Примеры:**
- Апрель 2026 → Март 2026
- Январь 2026 → Декабрь 2025

---

### 4.9 `goToNextMonth()`

**Назначение:** Переключение на следующий месяц

**Параметры:** нет

**Возвращает:** `void`

**Алгоритм:**
```javascript
const goToNextMonth = () => {
  setCurrentDate(
    new Date(currentDate.getFullYear(), currentDate.getMonth() + 1)
  );
};
```

**Примеры:**
- Апрель 2026 → Май 2026
- Декабрь 2026 → Январь 2027

## 11. Тестирование

### 11.1 Unit тесты (рекомендуется)

```javascript
import { render, screen, fireEvent } from '@testing-library/react';
import BookingCalendar from './calendar-component';

describe('BookingCalendar', () => {
  test('отображает текущий месяц', () => {
    render(<BookingCalendar />);
    const today = new Date();
    const monthName = monthNames[today.getMonth()];
    expect(screen.getByText(new RegExp(monthName))).toBeInTheDocument();
  });

  test('не позволяет выбрать прошедшую дату', () => {
    render(<BookingCalendar />);
    const yesterday = new Date();
    yesterday.setDate(yesterday.getDate() - 1);
    const button = screen.getByText(yesterday.getDate().toString());
    expect(button).toBeDisabled();
  });

  test('выделяет выбранную дату', () => {
    render(<BookingCalendar />);
    const tomorrow = new Date();
    tomorrow.setDate(tomorrow.getDate() + 1);
    const button = screen.getByText(tomorrow.getDate().toString());
    fireEvent.click(button);
    expect(button).toHaveStyle({ background: '#667eea' });
  });
});
```

## 12. Известные ограничения

### 12.1 Текущие ограничения

1. **Хардкод производственного календаря**
   - Праздники 2026 зашиты в код
   - Требуется обновление для 2027+
   - **Решение:** Вынести в API или конфиг файл

2. **Нет часовых поясов**
   - Всё в локальном времени пользователя
   - **Решение:** Использовать UTC + timezone конвертация

3. **Только русский язык**
   - Названия месяцев и дней на русском
   - **Решение:** i18n библиотека (react-i18next)

4. **Нет валидации диапазона**
   - Можно выбрать дату через 10 лет
   - **Решение:** Добавить minDate/maxDate props

### 12.2 Браузерная совместимость

✅ **Поддерживается:**
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

⚠️ **Частично:**
- IE 11 (требуется полифилл для Date методов)
