# Техническая документация: F8.1 - Страница подтверждения

**Версия:** 1.0  
**Компонент:** `ConfirmationPage`  
**Файл:** `confirmation-page.jsx`  

## Описание

Финальная страница после успешного бронирования встречи.

## Props

```javascript
{
  bookingData: {
    duration: number,           // Длительность (мин)
    date: string,              // Дата и время
    formData: {                // Данные клиента
      firstName: string,
      lastName: string,
      email: string,
      phone: string,
      telegram?: string,
      additionalParticipants?: string,
      questions?: string
    }
  }
}
```

## Отображаемая информация

### Детали встречи
- 📅 Дата и время
- ⏱️ Длительность
- 👤 Имя участника

### Контакты
- ✉️ Email
- 📞 Телефон
- 💬 Telegram (если указан)

### Дополнительно
- Дополнительные участники (если есть)
- Вопросы для обсуждения (если есть)

## Что дальше?

1. Письмо с подтверждением на email
2. Ссылка МТС-Линк за 1 час до встречи
3. Напоминание в Telegram (если указан)

## Действия

- **Добавить в календарь** - сохранить в Google Calendar
- **На главную** - вернуться на главную страницу

## Анимации

- scaleIn - иконка успеха
- slideUp - карточки (с задержкой)

## UI/UX

- Большая зелёная галочка ✓
- Все данные бронирования
- Пошаговые инструкции
- Понятные действия
