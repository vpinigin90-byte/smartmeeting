# Техническая документация: F3.1 - Форма бронирования

**Версия:** 1.0  
**Компонент:** `BookingForm`  
**Файл:** `booking-form.jsx`  

## Обязательные поля

1. **Имя** (firstName)
   - Валидация: минимум 2 символа
   - Icon: User

2. **Фамилия** (lastName)
   - Валидация: минимум 2 символа
   - Icon: User

3. **Email**
   - Валидация: регулярное выражение `/^[^\s@]+@[^\s@]+\.[^\s@]+$/`
   - Icon: Mail

4. **Телефон** (phone)
   - Валидация: 11 цифр, начинается с 7 или 8
   - Форматирование: +7 999 123-45-67
   - Icon: Phone

## Props

```typescript
{
  onSubmit?: (data: FormData) => void;
  selectedDate?: string;
  selectedDuration?: number;
}
```

## State

- `formData` - данные полей
- `errors` - ошибки валидации
- `touched` - touched поля

## Функции валидации

### validateEmail(email)
Проверяет формат email через regex

### validatePhone(phone)
- Убирает всё кроме цифр
- Проверяет 11 цифр
- Проверяет начало с 7 или 8

### formatPhone(value)
Форматирует телефон в вид: +7 999 123-45-67

### validateField(name, value)
Универсальная валидация по имени поля

## Поведение

- **Inline валидация** при потере фокуса
- **Real-time валидация** после первого blur
- **Submit disabled** пока форма невалидна
- **Focus states** с синей рамкой
- **Error states** с красной рамкой и сообщением

## Accessibility

- Иконки для визуальной идентификации полей
- Красные звёздочки (*) для обязательных полей
- Подсказки под полями
- Focus states
- AutoComplete атрибуты

## Интеграция

```jsx
<BookingForm
  selectedDuration={60}
  selectedDate="15 апреля 2026"
  onSubmit={(data) => console.log(data)}
/>
```
